Every improvement, fix, and new feature — in order, with dates.

April 2026
v1.0

Monte Carlo Risk Engine & S-Curve View

Major analysis release

  • NEW Proper PERT Beta Monte Carlo engine — 10,000 iterations using Marsaglia-Tsang gamma sampling
  • NEW Risk correlation groups — risks in the same group fire together in simulation
  • NEW Monte Carlo S-Curve view with P50/P80/P90 lines, histogram, and hover tooltips
  • FIX Critical path toggle now uses proper CPM float engine — consistent with Float Analysis tab
  • FIX Canvas ResizeObserver infinite loop resolved — no more jumping on hover
  • UX Shape-aware selection rings — diamonds, circles, stars no longer show ugly bounding boxes
April 2026
v0.9

Calendar System

Working day scheduling

  • NEW Visual calendar builder with month-grid day picker — click days to mark non-working
  • NEW Calendar assignment per activity — 7-day and 5-day Standard built in
  • NEW Scheduler stretches/shrinks bars based on calendar — switch 7-day→5-day and re-schedule correctly
  • NEW Duplicate calendar button

Major UX overhaul + commercial infrastructure
