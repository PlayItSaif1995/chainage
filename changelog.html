April 2026
v1.2

Guided Tour, Draw-to-Create & Grid Overhaul

UX and onboarding release

  • NEW9-step interactive guided tour with spotlight highlights — launches on first open or via the ? menu
  • NEWDraw-to-create on canvas — press D or click ✏ Draw, then drag diagonally to place a new activity at exact dates and chainages
  • NEWGrid view column tiers — 9 core columns visible by default, advanced columns hidden; use the ⊞ Columns picker to toggle
  • NEWColour palette picker in grid — click any Colour cell to pick from 28 infrastructure-friendly swatches
  • FIXBlack theme button colours — --btn-primary-bg variable ensures buttons use blue in all themes
  • FIXScheduler resets successors to originalStart before forward pass — switching calendars no longer leaves downstream activities stranded
  • FIXSchedule confirmation modal warns user before running CPM forward pass
  • UX? button now opens help menu with Documentation, Guided Tour, and Keyboard Shortcuts
  • UXPopout window tooltips correct for all 7 views including float labels, zone clashes, and Monte Carlo P-bands

v1.1

Pop-out Live Preview & Fullscreen Tooltips

Output and analysis release

  • NEWPop-out window (⧉) — opens live chart preview in a separate browser window, auto-updates on every change
  • NEWAll 7 views available in pop-out
  • FIXFullscreen tooltips now show float labels, zone clashes, Monte Carlo P-bands and cost breakdowns correctly
  • FIXCalendar revert bug — switching from 5-day back to 7-day correctly restores original end date
  • FIXCPM forward pass — activities no longer move backward when constraint already satisfied

Monte Carlo Risk Engine & S-Curve View

Major analysis release

  • NEWEditable grid view (Handsontable) with Excel paste, fill-handle, sort and filter
  • NEWGroups column in grid — type "Zone A / Drainage" to auto-create and assign groups
  • FIXCritical path toggle uses proper CPM float engine
  • UXShape-aware selection rings for all activity shapes
  • UXRight-click context menu — add, edit, delete, link activities on canvas
